# Conditions app

## Server
go into server directory

### Build
`yarn`

### Run

`yarn start`

## Client
go into client directory

### Build
`yarn`

### Run
`yarn start`

This will run the client  
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Run tests
`yarn test`
